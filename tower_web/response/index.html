Types and traits for responding to HTTP requests.

The Response trait is implemented by types that can be converted to an HTTP response. Resource methods must return futures that yield types implementing Response.

Currently, Response implementations are provided for the following types:

  • String
  • &'static str
  • http::Response
  • serde_json::Value
  • tokio::fs::File

More implementations can be added by submitting a PR.

Also, Response can be implemented for custom types by using the [derive(Response)] proc macro. See library level documentation for more details.
